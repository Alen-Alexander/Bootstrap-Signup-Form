# Bootstrap-Signup-Form
Professional Bootstrap 4 signup form with responsive grid, form validation UX, and modern design patterns. Job-ready frontend demo

Features
- **Bootstrap 4 Grid**: Responsive 2-column layout (`col-12 col-md-6`)
- **Flexbox Utilities**: `d-flex justify-content-center align-items-center`
- **Form UX**: Password confirmation, terms checkbox, validation states
- **Viewport Design**: Full-height `vh-100` + `container-fluid`
- **Modern Styling**: Custom primary theme + light input backgrounds
